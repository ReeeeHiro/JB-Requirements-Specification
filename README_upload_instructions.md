# Upload instructions

Upload these files to the root of the GitHub repository:

- `index.html`
- `index_en.html`
- `index_zh-hant.html`
- `index_zh-hans.html`
- `mainlogo_backclear.png`

After upload, open:

- Japanese: `/`
- English: `/index_en.html`
- Traditional Chinese: `/index_zh-hant.html`
- Simplified Chinese: `/index_zh-hans.html`

Language buttons are added to the sticky header.
